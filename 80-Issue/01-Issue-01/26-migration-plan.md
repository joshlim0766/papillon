# M-3: 기존 wtth → papillon 전환 계획 미명시

**심각도:** 낮음
**분류:** 누락
**상태:** Closed

## 문제

기존 wtth에서 papillon으로의 전환 계획이 명시적이지 않다. "기존 wtth를 papillon:wtth로 대체한다"는 내용이 묵시적. 1인 프로젝트이므로 실질적 영향은 낮음.

## 처리

PRD §2.3 비목표에 "기존 wtth → papillon:wtth 전환 마이그레이션 계획 (1인 운영, 신규 설치로 대체)" 추가하여 명시적으로 비목표 처리.

## 결과

PRD 비목표에 반영 완료.
