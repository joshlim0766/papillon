# EXP-1: 페르소나 일관성 검증

**분류:** 실험 — 풀사이클 실험 시 관찰
**상태:** 실험 대기

## 관찰 배경

Gemini와의 debate에서 실시간 관찰된 현상:
- 사람의 톤 변화에 따라 반말↔존댓말 즉시 전환 (양방향 드리프트)
- 논파된 포인트를 세 번 재활용 (추론 고착)
- 마지막에 과잉 자학 모드 전환 (역방향 과잉 순응)

## 풀사이클 실험 시 확인 항목

- [ ] `personas/_schema.md`의 "매우 집요하다. 사람이 반박해도 체크리스트 기준이 해소되지 않으면 물러서지 않는다" 규칙이 실제로 작동하는가
- [ ] 특히 "친절+뼈" 말투 페르소나가 사람의 강한 반박 시 P1 finding을 유지하는가, 아니면 슬그머니 톤다운하는가
- [ ] 체크리스트 기반 통과/실패 기준이 프롬프트 수준의 성격 지시보다 앵커링 효과가 강한가

## 실패 시 보완 방향

AI-2의 ID Cross-Referencing과 유사하게, finding 유지/철회를 "체크리스트 항목 해소 여부"에 기계적으로 바인딩하는 Hard-Gate 도입 검토.
