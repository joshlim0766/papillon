# 1-2: M→L 승격 안전망이 Phase 2 상세에 누락

**심각도:** 중간
**분류:** 구조적 일관성
**상태:** Open

## 문제

다이어그램에 "리뷰 1라운드 후 wtth이 건수/심각도/패턴 분석하여 태스크 크기 경고 가능 (M→L 승격 안전망)"이 명시되어 있다. Phase 2 상세 섹션에서는 이 경고 메커니즘 언급이 없다. M→L 승격은 공통 규칙 2.1의 승격 테이블에만 한 줄로 적혀 있고, wtth 1라운드 분석 결과를 받아 승격을 판단하는 흐름이 Phase 2 상세에 기술되어 있지 않다.
