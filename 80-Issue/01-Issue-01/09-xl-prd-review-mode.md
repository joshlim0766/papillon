# 2-3: XL의 Phase 2가 PRD 리뷰 모드로 동작하는 분기 없음

**심각도:** 중간
**분류:** 논리적 공백
**상태:** Closed

## 문제

Step 2에서 XL은 "PRD 작성 → Phase 2(wtth PRD 리뷰 모드)"로 진행한다. Phase 2 상세는 "설계 리뷰 모드"로만 기술되어 있고, PRD 리뷰 모드에서의 동작(ADR 승격이 PRD 리뷰에서도 발생하는지 등)은 언급이 없다. XL의 Phase 2가 PRD 리뷰 모드로 동작한다는 분기가 Phase 2 상세에 기술되어야 한다.

## 논의

- PRD는 범위·제약 설정 문서이므로 구조적 결정(ADR)이 나올 단계가 아님
- M→L 승격도 XL에는 해당 없음 (이미 최상위)
- finding 수용 시 PRD 자체를 수정하는 것으로 충분

## 처리

Phase 2 섹션을 리뷰 모드별로 구조화:
- 공통 흐름 (wtth 호출 → 프로세스 수행 → RDR 생성)
- 설계 리뷰 모드 (M 체급) 추가 흐름: M→L 승격 판단 + ADR 승격
- PRD 리뷰 모드 (XL 체급) 추가 흐름: ADR 승격 없음, finding 수용 시 PRD 수정

## 결과

papillon 설계서 §3.2를 리뷰 모드별 분기 구조로 개편 완료.
