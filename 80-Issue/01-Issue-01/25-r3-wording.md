# X-4: R3 제약 "기본/우선" vs common-spec "대신" 미세 용어 차이

**심각도:** 낮음
**분류:** 용어 일관성
**상태:** Closed

## 문제

PRD R3: "common-spec §11 기본, 사용자 doc-standard 우선". common-spec §11.6: "사용자 doc-standard가 있으면 §11.1~11.5 대신 사용자의 규칙을 따른다." 의미는 일치하나 용어가 다름.

## 처리

PRD R3 문구를 common-spec과 일치시킴: "기본, 사용자 doc-standard 우선" → "기본, 사용자 doc-standard가 있으면 대신 적용".

## 결과

용어 통일 완료.
