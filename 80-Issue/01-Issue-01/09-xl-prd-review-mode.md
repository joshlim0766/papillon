# 2-3: XL의 Phase 2가 PRD 리뷰 모드로 동작하는 분기 없음

**심각도:** 중간
**분류:** 논리적 공백
**상태:** Open

## 문제

Step 2에서 XL은 "PRD 작성 → Phase 2(wtth PRD 리뷰 모드)"로 진행한다. Phase 2 상세는 "설계 리뷰 모드"로만 기술되어 있고, PRD 리뷰 모드에서의 동작(ADR 승격이 PRD 리뷰에서도 발생하는지 등)은 언급이 없다. XL의 Phase 2가 PRD 리뷰 모드로 동작한다는 분기가 Phase 2 상세에 기술되어야 한다.
