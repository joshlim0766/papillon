# I-2: 파일명 표기 혼재

**심각도:** 낮음
**분류:** 용어 일관성
**상태:** Closed

## 문제

PRD 섹션 4에서 `papillon:inquisition.md` 형식(콜론 포함)으로 표기하나, 실제 설치 경로(common-spec §9)에서는 `inquisition.md`(콜론 없음). 혼동 여지.

## 처리

PRD 섹션 4 산출물 테이블 + Output 목록에서 콜론 제거: `papillon:inquisition.md` → `inquisition.md` 등.

## 결과

PRD 파일명 표기를 실제 설치 경로와 일치시킴.
