# 03-전문가 풀 제한 메커니즘

**상태:** Settled

## 확인된 사항
- 설계 리뷰의 6인(ARCH/BE/FE/SRE/SEC/PO)은 "뽑아 쓰는 풀"이지 전원 투입이 아님
- 기본 풀 상한 3인: 오케스트레이터가 3인 초과를 추천하려면 사람 확인 필요
- excluded_experts: 사람이 제외한 전문가를 리뷰 세션(RDR 1개 단위) 동안 유지. 해당 리뷰 내 재추천 금지
- 클로드 세션이 아닌 리뷰 세션 기준으로 제외가 유지되어야 함 (다중 세션 리뷰 시 영속성 필요)
- 기존 모드별 기본 풀(PRD 3인, ADR 2인, 운영절차 2인)은 이미 3인 이하 — 설계 리뷰만 예외적으로 6인 풀
