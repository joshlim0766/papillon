# X-2: 설계서 Context Related에 shackled 설계서 링크 부재

**심각도:** 중간
**분류:** 참조 일관성
**상태:** Closed

## 문제

03-design-papillon.md의 Context Related에 `04-design-wtth`, `05-design-inquisition`은 있으나 shackled 관련 설계서 링크가 없다. 아직 shackled 설계서가 미작성(인터뷰만 완료)이므로 현재는 의도적이나, 설계서 작성 시 누락하지 않도록 기록.

## 논의

설계서는 아직 없지만 인터뷰 완료 상태이므로 index 링크로 연결. 설계서 작성 시 경로 업데이트 필요.

## 처리

Context Related에 `[06-shackled/00-index.md](./06-shackled/00-index.md)` 추가.

## 결과

papillon 설계서 Context Related에 shackled 링크 반영 완료.
