# 3-4: shackled "페어 프로그래밍 스킬" 명칭 vs 실제 역할 괴리

**심각도:** 낮음
**분류:** 용어/참조 일관성
**상태:** Closed

## 문제

06-shackled/00-index.md 제목이 "페어 프로그래밍 스킬"이고, 인터뷰 summary에서도 동일. 그러나 MVP에서는 기본 모드(자동)만 구현하고 페어 모드는 Deferred. 실제 역할은 "구현 전문 스킬"에 더 가깝고, "페어 프로그래밍"은 향후 확장 모드 중 하나. 초기 진입자에게 혼동 가능.

## 논의

index 제목은 현재 역할에 맞게 변경. 인터뷰 summary는 당시 기록이므로 보존.

## 처리

`06-shackled/00-index.md` 제목: "페어 프로그래밍 스킬" → "구현 스킬".

## 결과

index 제목 변경 완료.
